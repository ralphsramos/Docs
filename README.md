# Coletor 2001

Aplicativo Android de coleta e **impressão de etiquetas** para coletores e
celulares, com sincronização de catálogo e **atualização online (OTA)**.

## 📦 Download / atualização do app

As versões do app (APK) ficam em **[Releases](../../releases)**. Cada release
traz o **APK assinado** e o **SHA-256**, que o app confere antes de aplicar a
atualização.
