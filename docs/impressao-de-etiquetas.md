# Impressão de etiquetas

> 🚧 **Em construção.** Fluxo principal do app — completar com exemplos.

Imprime **etiquetas de preço** a partir da leitura do produto.

## Fluxo básico

1. Abra o módulo **Impressão**.
2. **Bipe** o código do produto (câmera ou leitor) — ou digite.
3. Escolha o **preço a imprimir** quando houver mais de uma opção
   (tabela / promoção / à vista).
4. Ajuste a **quantidade de cópias** e **imprima**.

## Impressão em lote (lista de produtos)

Dá para montar uma **lista** de produtos (por código, EAN ou filtros) e imprimir
todas as etiquetas de uma vez.

## Impressoras

O app imprime por:

- **Rede (TCP)** — impressora com endereço na rede;
- **Bluetooth** — impressora portátil pareada;
- **Compartilhamento** — impressora compartilhada no servidor.

🚧 *A detalhar: cadastrar a impressora, escolher o layout da etiqueta e ajustar
o tamanho.*
