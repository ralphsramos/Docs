# Coletas e inventário

> 🚧 **Em construção.**

## Coletas avulsas

Bipagem **livre** de produtos para conferência posterior. Cada leitura entra numa
lista local; ao finalizar, a coleta é enviada e fica disponível no painel.

- Bipe por câmera ou leitor.
- Informe a **quantidade** e, se quiser, uma observação.
- **Finalize** para enviar.

## Inventário

Contagem organizada por **setor** e **área**.

1. Baixe o inventário disponível para o dispositivo.
2. Selecione o **setor** e a **área**.
3. **Conte** (bipe + quantidade; dá para registrar também a quantidade avariada).
4. **Finalize a contagem** da área para transmitir.

As leituras ficam **salvas no aparelho** até o envio — não se perdem se o app
fechar.
